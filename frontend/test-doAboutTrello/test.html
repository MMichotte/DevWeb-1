Voici les règles du jeu du pendu :

Le but est de trouver le mot avant d'être tout à fait pendu.
A chaque fois qu'une lettre n'apparait pas dans le mot, le petit bonhomme est de plus en plus pendu.
Si vous trouvez le mot avant que le bonhomme soit pendu, vous avez gagné !
